Integration Examples

1. Basic HTML Integration

<!-- Include the JavaScript file --> <script src="booking-hotel-button.js"></script> <!-- Create a container for the button --> <div id="booking-container"></div> <script> // Set your search parameters searchQuery.destination = 'London, UK'; searchQuery.checkinDate = '2024-03-15'; searchQuery.checkoutDate = '2024-03-18'; // Initialize the button initializeBookingButton('booking-container'); </script>

2. Dynamic Integration with Options

<script> // Initialize with options initializeBookingButton('booking-container', { destination: 'Tokyo, Japan', checkinDate: '2024-04-01', checkoutDate: '2024-04-05' }); </script>

3. Manual Button Creation

<script> // Set parameters searchQuery.destination = 'New York, USA'; searchQuery.checkinDate = '2024-05-10'; searchQuery.checkoutDate = '2024-05-15'; // Create and append button manually const button = createBookingButton(); document.body.appendChild(button); </script>

4. URL Construction Only

<script> // Just get the URL without creating a button const url = constructBookingUrl( 'Barcelona, Spain', '2024-06-01', '2024-06-05' ); console.log(url); // Logs the complete Booking.com URL </script>

URL Format

The generated URL follows this format:

https://www.booking.com/searchresults.html?ss=[LOCATION]&checkin_year=[YEAR]&checkin_month=[MONTH]&checkin_day=[DAY]&checkout_year=[YEAR]&checkout_month=[MONTH]&checkout_day=[DAY]

Example:

https://www.booking.com/searchresults.html?ss=Paris%2C%20France&checkin_year=2024&checkin_month=3&checkin_day=15&checkout_year=2024&checkout_month=3&checkout_day=18
